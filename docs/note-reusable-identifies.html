Use of Icons Provided by Firefox

  • Draft Note - January 27, 2010
  • by Marcio Galli

During the development of the ExpressionWidgets I feel I needed to use icons - like warnings, question marks or other built in meaninful iconsets. This action came out possibly inspired by the original samples and also motivated myself to think about an extended API function that could provided this effect to a further degree. After some research using the Mozilla source code view tool ( http://mxr.mozilla.org ) additional icons where found:

  • chrome://global/skin/notification/warning-icon.png
  • chrome://global/skin/notification/question-icon.png
  • chrome://global/skin/media/throbber.png
These icons were found when visiting the default theme directory which can be visualized using MXR.
